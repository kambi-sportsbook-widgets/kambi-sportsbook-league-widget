# league-widget

Displays the league teams with statistics, with data from statistics api

## Configuration example:

__`client-widgets.js`__

```json

...
{
    "order": 1,
    "widgetId": "League widget",
    "args": {
        "updatedTime": '08:00 GMT',
        "style": "padding-top: 12px;"
    }
},
...

```
